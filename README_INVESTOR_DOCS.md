  2. Muokkaa sarakkeet omilla luvuillasi

KRIITTISET HAVAINNOT
─────────────────────────────────────────────────────────────────────────────────────

✅ VAHVUUDET:
   • Digitaalinen = skalautuva ilman lineaarista henkilöstönkasvua
   • Korkea voittomarginaali = kestävä kasvu
   • Nopea ROI = helppo sijoituksen hyväksyntä
   • Selkeä markkinarako (Pomminvarmat ei digitaalinen)
   • Kansainvälistymispotentiaali (Pohjoismaat → Eurooppa)

⚠️ RISKIT:
   • Asiakkaiden hintasensitiivisyys (ratkaisu: asteittainen nousu)
   • Kilpailun vastaus (ratkaisu: parempi palvelu)
   • Henkilöstön saatavuus (ratkaisu: paremmat palkat + bonus)

🎯 KRIITTINEN MENESTYSTEKIJÄ:
   • Ensimmäisen 6 kuukauden aikana saavuttaa 5+ muuttoa/viikko
   • Siitä eteenpäin kasvu on helpompaa ja automaattisempaa


SEURAAVAT ASKELEET
─────────────────────────────────────────────────────────────────────────────────────

1. Presentoi PDF investoijille (MUUTTOKONE_INVESTOR_DECK.pdf)
2. Vastaanota Q&A - anna FINANCIAL_DATA.csv tarkeemmiksi laskelmiksi
3. Neuvottele investointisumma ja -ehdot
4. Aloita MVP-toteutus (kalkulaattori + palveluntarjoajien verkosto)
5. Markkinoi aggressiivisesti ensimmäisen kuukauden aikana
6. Osoita proof-of-concept (5+ muuttoa/viikko) → helpommin sijoitus
