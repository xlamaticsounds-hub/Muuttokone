# Johdanto

Tervetuloa **Muuttokone.fi** -alustan tekniseen dokumentaatioon. Tämä sivusto on rakennettu modernilla Next.js 16 -arkkitehtuurilla tarjoamaan salamannopean käyttökokemuksen sekä asiakkaille että ylläpitäjille.

## Alustan peruspilarit

1.  **Dynaaminen sisältö (LEGO-lähestymistapa)**: Sivuston sisältö on täysin hallittavissa tietokannan kautta, mikä mahdollistaa tekstien ja kuvien muokkaamisen ilman koodimuutoksia.
2.  **AI-integraatio**: Alusta hyödyntää OpenAI:n ja Googlen kielimalleja sisällöntuotannossa ja asiakaspalvelussa.
3.  **Skaalautuva infrastruktuuri**: Käytämme Google Cloud Platformin (GCP) serverless-palveluita (Cloud Run, Cloud SQL, GCS), jotka tarjoavat yritystason luotettavuutta.
4.  **Tietoturva**: Kaikki salaisuudet on suojattu ja API-rajapinnat on turvattu automaattisella tunnistautumisella ja nopeusrajoituksilla.

## Teknologiavalinnat

- **Framework**: Next.js 16 (App Router)
- **Database**: PostgreSQL & Prisma ORM
- **Styling**: Tailwind CSS v4
- **Auth**: NextAuth.js v4
- **Storage**: Google Cloud Storage
- **Deployment**: GCP Cloud Run & Cloud Build

Valitse aihe vasemmasta sivupalkista aloittaaksesi.
